A dynamic, responsive News Aggregator Web Application that fetches and displays the latest headlines across various categories like Science, Finance, Politics, and Sports.

🚀 Features
Real-time News Fetching: Stay updated with the latest headlines from around the globe.

Category Filtering: Easily navigate between different topics including IPL (Sports), Finance, Politics, and Science.

Search Functionality: Find specific news articles using the integrated search bar.

Responsive Design: Optimized for a seamless experience on desktops, tablets, and mobile devices.

Clean UI: Minimalist interface focused on readability and quick information consumption.

🛠️ Tech Stack
Frontend: HTML5, CSS3, JavaScript (ES6+)

Styling: Custom CSS (with Flexbox/Grid for layout)

API: NewsAPI (or specify the API you are using)
